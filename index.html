Start a New Project

  1. Open VS Code
  2. Select 'File'.
  3. Select 'Open Folder...'.
  4. Click on the 'Create Folder' icon in the the top right corner of the dialogue.
  5. Enter a name for the folder in the 'folder Name' field and click 'Create'.
  6. Click 'OK'.
  7. Select 'File' and then select 'New File'.
  8. From the 'EXPLORER' navigation pane click on the 'New File' icon.
    • If the 'EXPLORER' navigation pane is not visible select the 'Explorer' icon form the left hand pane or press 'CTRL' + 'Shift' + 'E'.
  9. Enter a name for the file including the file extension i.e. index.html and press 'Enter'.

Create a New Project with GitHub Version Control

  1. Log into GitHub.
  2. Create a 'New' repository.
  3. In the 'Repository name' field enter the project name.
  4. Click 'Create Repository'.
  5. Copy the 'URL' from the browser address bar.
  6. From within 'VS Code' press 'CTRL' + 'SHIFT' + 'P'.
  7. Type 'Git: clone'.
  8. Paste the URL which was copied from the browser address bar in step 5, and press 'Enter'.
  9. Select the location where you would like to save your project, and click the 'Select Repository Location' button.
  10. Start working on your project, creating Files/Folders, saving your project as you go along.
  11. Select the 'Source Control' icon from the 'Activity Bar' on the left hand side.
  12. Enter a comment in the 'message' field in the 'Source Control' 'Side Bar'.
  13. Click the 'Commit' button in 'Source Control' 'Side Bar'.
  14. Click the ellipsis (...) in the 'Source Control' 'Side Bar' and select 'Push' from the menu.
  15. Enter your GitHub user name into the 'Username' field and press 'Enter'.
  16. Enter your GitHub passwword into the 'Password' field and press 'Enter'.
  17. Check your GitHub page to ensure your initial push has completed and your files are present.
